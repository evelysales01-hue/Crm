# CRM Móveis Planejados

## Como rodar

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

Acesse o link que o Vite mostrar no terminal.

- email: admin@empresa.com
- senha: 123456

## Recursos da versão 1
- Login admin/vendedor
- Cadastro de clientes
- Cadastro de produtos/ambientes por cliente
- Ticket total automático
- Aniversariantes do dia
- Mensagem automática de aniversário
- Relatório base por vendedores via API
